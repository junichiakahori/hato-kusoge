■ スライド 4：テクノロジースタック
------------------------------------------------------------------------
【スライド内のテキスト】
🛠️ テクノロジースタック
・フロントエンド：HTML5 Canvas, Vanilla CSS (ネオン・ダークテーマ), JavaScript
・サウンド：Web Audio API によるリアルタイムシンセサイザー
  ★ 外部のMP3などの音声ファイルは一切不要！ブラウザ上で波形を合成
・サーバー：Node.js + Express (ローカルでのランキングAPI保存)
・データベース：Supabase (オンラインでのグローバルランキング連携)

【発表用台本（スピーカーノート）】
「技術構成です。フロントエンドはライブラリを極力使わず、軽量に動作するようHTML5 Canvasと素のJavaScript、CSSで構築しています。
特に音響面では、Web Audio APIを使ったリアルタイムシンセサイザーを実装しており、羽ばたき音やゲーム音、BGMなどをすべてコード上で動的に生成しているため、外部音源ファイルの読み込みが不要で非常に軽量です。
また、ローカルとオンライン（Supabase）の両方のランキング機能に対応しています。」
