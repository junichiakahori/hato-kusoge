### v1.3.0 (2026/06/09)
- **ゲーム要素の拡張**
  - 敵キャラクター（カラス、猫）に対してハトのフンを落として撃墜できる機能を追加（撃墜時はボーナススコア獲得、羽が舞い散るパーティクル演出）。
  - 撃墜したカラス（30%の確率）や猫（50%の確率）から回復アイテム「ライフ（ハート）」がドロップする仕組みを導入。
  - 徐々に漂いながら明滅し、一定時間で消滅するライフ回復アイテムの挙動を追加。
- **グラフィック・演出**
  - 朝 ➔ 昼 ➔ 夕方 ➔ 夜の24時間で空や背景が徐々に変化する「動的背景サイクル」システムを導入。
  - 夜間において、街灯からの動的な光のコーン表示および電線への環境光のグロー反射を描画し、夜間の飛行ルートの視認性を向上。
- **操作性・システム改善**
  - 電線に止まって休んでいる状態（Perched/Resting）でもフンを投下できるようにし、そのフンの衝突判定を有効化。
  - 電線着地時の当たり判定を電線のビジュアルなたるみ（たるんだ曲線）に一致させ、着地時・離脱時の不具合を修正。
  - 高リフレッシュレートのモニター環境でゲームスピードが上がってしまう問題を解消するため、deltaTimeを用いた 60 FPS 上限（ゲームループのウェイト処理）を導入。
  - UIやショップ、説明文等において、ゲーム内アイテムの呼称を「ブレッドクラム（パンくず）」から「エサ（food / 🍞）」に統一。
  - モバイル操作用の羽ばたき/フン落としボタンを拡大し、位置を40px上部に移動、および羽ばたきボタンの背景を透過させて画面を広く見せるように調整。

### v1.1.0
- **Supabase グローバルランキング接続**
  - 独自の Supabase プロジェクトに対応するための API キーの接続設定と自動 URL サニタイズ処理（末尾スラッシュの補正や `/rest/v1` パスの自動補完）を追加。
  - 以前使用していた Render API から Supabase REST API を用いた永続的な無料ランキングサーバー構成へ移行。
  - ランキングデータが存在しない際の初期ダミーデータを削除し、誰も登録していない状態用の「最初の登録者になろう！🏆」表示（空のUIステート）を実装。
  - ネットワークエラー等の障害発生時に備え、ブラウザの LocalStorage を利用したオフラインでのランキングフォールバック機能を実装。
