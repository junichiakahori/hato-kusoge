# ハトの糞ゲー 🕊️

### 都会を飛び回る、爽快フン爆撃アクション！

---

## 🎮 プロジェクト概要

「クソゲー」をテーマにした、ユーモラスで爽快なプレイ感を提供するPC・モバイル向けWebアクションゲームです。

- **目的**：ハトを操作してエサを食べ、障害物を避けながら、地上の人々や車にフンを命中させてハイスコアを目指します。
- **体験**：簡単操作で爽快感を得られる、レトロでネオンなアーケード風デザイン。
- **特徴**：AIアシスタントと人間が強力にタッグを組んで開発されました。

---

## ⚡ 主なゲームシステム

1. **爽快な飛行物理 ＆ 着地システム**
   - 羽ばたきの浮遊感や慣性をシミュレーション。
   - 電線や電柱に着地してスタミナとフンの弾数を高速回復！
2. **フン爆撃ギミック（💩）**
   - 歩行者や走る車にフンを落とし、楽しい悲鳴リアクションと共に高得点。
3. **ハト・ショップ（育成とカスタマイズ）**
   - エサ（パン）を消費してスタミナ・弾数・速度を強化。
   - 特徴的なスキン（ノーマル、ゴールデン、サイバー、シャドウ）の解放。

---

## 🛠️ テクノロジースタック

外部の重いライブラリを最小限に抑え、軽量・高速に動作するよう構築されています。

- **フロントエンド**：HTML5 Canvas, Vanilla CSS (ネオンテーマ), JavaScript (ES6+)
- **サウンドシステム**：**Web Audio API によるリアルタイムシンセサイザー**
  - BGMや効果音用の音声ファイルは一切不要！ブラウザ上で動的に音波を合成・生成。
- **バックエンド**：Node.js + Express (ローカルランキング)
- **データベース**：Supabase (オンラインランキング連携)

---

## 🤖 AI (Google Antigravity) との共同開発

このプロジェクトは、AIアシスタントとのペアプログラミングにより非常に短い期間で高品質に開発されました。

- **アルゴリズム実装**：Web Audio APIでの効果音合成、ハトの羽ばたき物理演算、当たり判定ロジックの構築。
- **UI・デザインの調整**：CSSによるネオン調ダークテーマの最適化、レスポンス調整。
- **ドキュメント・動画連携**：READMEへの自動インライン動画プレビューの設定やGit管理の自動化。
