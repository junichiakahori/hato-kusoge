ハトの糞ゲー 🕊️

都会を飛び回る、爽快フン爆撃アクション！

Developed in collaboration with Google Antigravity

🎮 プロジェクト概要

作ってしまった... 究極のクソゲー！

  • ユーモラスで爽快なプレイ感を提供するWebアクション
  • エサを食べ、障害物を避け、地上の人々にフンを落とす！
  • レトロなネオン調のゲームセンター風デザイン

⚡ ユニークなゲームシステム

  • 飛行物理 ＆ 電線への着地 羽ばたきの浮遊感や慣性をシミュレート。電柱に着地して高速回復！
  • 楽しいフン爆撃ギミック ターゲットにフンが当たると、面白い叫び声とともに高得点を獲得。
  • ハト・ショップでの育成 集めたパンでハトを強化。ゴールデンやサイバーなどのスキンも解放！

🛠️ 音声ファイル一切不要！

Web Audio API による 動的シンセサイザー

  • BGMや効果音（羽ばたき音、ヒット音）はすべてコードで生成
  • 外部アセットのロードが無いため、一瞬で超軽量に起動
  • フロントエンドは Canvas とピュアな JavaScript のみで高速動作

🤖 AIとの爆速共同開発

Google Antigravity とのペアプログラミング

  • 音響エンジニアリング：シンセサイザーの合成ロジック構築
  • ゲーム物理：慣性のある羽ばたき挙動、着地判定アルゴリズム
  • デザイン最適化：ネオン調ダークテーマのビジュアル調整
