ハトの糞ゲー 🕊️

都会を飛び回る、爽快フン爆撃アクション！

Developed in collaboration with Google Antigravity

🎮 プロジェクト概要

  • 「クソゲー」をテーマにした、ユーモラスで爽快なプレイ感を提供するPC・モバイル向けWebアクションゲーム。
  • ハトを操作してエサを食べ、障害物を避けながら地上の人々にフンを落としてハイスコアを目指す！
  • レトロでネオンなアーケード風デザイン。

⚡ 主なゲームシステム

  • 飛行物理 ＆ 着地システム：羽ばたきの浮遊感を物理シミュレート。電柱や電線に着地してスタミナと弾数を高速回復！
  • フン爆撃ギミック（💩）：歩行者や車にフンを命中させ、楽しいリアクションと高得点を獲得。
  • ハト・ショップ：エサを消費してハトを強化。カスタムスキン（ノーマル、ゴールデン、サイバー、シャドウ）をアンロック。

🛠️ テクノロジースタック

  • フロントエンド：HTML5 Canvas, Vanilla CSS (ネオンテーマ), JavaScript
  • サウンドシステム：Web Audio APIによるリアルタイムシンセサイザー（外部の音声ファイル不要）
  • ランキング：Node.js + Express (ローカル) / Supabase (オンライン)

🤖 AIとの共同開発

Google Antigravity とのペアプログラミングにより高速開発！

  • Web Audio APIによる動的な効果音生成ロジックの構築
  • ハトの羽ばたき物理演算や衝突判定アルゴリズムの調整
  • CSSによるネオン調ダークテーマのビジュアル最適化
