ハトの糞ゲー 🕊️

都会を飛び回る、爽快フン爆撃アクション！

Developed in collaboration with Google Antigravity

🎮 プロジェクト概要

  • 「クソゲー」のテーマにふさわしい、ユーモラスなWebアクションゲーム
  • ハトを操作し、障害物を避けながら地上の人々にフンを落とす！
  • レトロでネオンなアーケード風のビジュアルデザイン

⚡ 主なゲームシステム

  • 飛行物理 ＆ 着地システム 羽ばたきの浮遊感や慣性をシミュレート。電線や電柱に着地して高速回復！
  • フン爆撃ギミック（💩） 歩行者や車にフンを命中させ、面白いリアクションと高得点を獲得。
  • ハト・ショップ（育成 ＆ カスタマイズ） エサで能力を強化。ゴールデンやサイバーなどのユニークなスキンを解放。

🛠️ テクノロジースタック

  • フロントエンド：HTML5 Canvas / Vanilla CSS / JavaScript (ES6)
  • サウンド：Web Audio API によるリアルタイムシンセサイザー ★ 音声ファイル一切不要！ブラウザ上で動的に効果音やBGMを合成
  • サーバー ＆ データベース：Node.js + Express / Supabase

🤖 AIとの共同開発

Google Antigravity とのペアプログラミングによる高速開発

  • 効果音合成アルゴリズム：Web Audio APIでの動的な効果音・BGMのコード生成
  • ゲーム物理の調整：慣性を伴う飛行挙動や、電柱への着地当たり判定の最適化
  • ビジュアル ＆ UI：ネオン調ダークテーマCSSとレスポンス画面調整
