ハトの糞ゲー 🕊️

都会を飛び回る、 爽快フン爆撃アクション！

Developed in collaboration with Google Antigravity

🎮 プロジェクト概要

作ってしまった... 究極のクソゲー！

  • 「クソゲー」をテーマにした、 ユーモラスなWebアクション
  • エサを食べ、 障害物を避けて 地上の人々にフンを落とす！
  • レトロでネオンな ゲームセンター風デザイン

⚡ ユニークなゲームシステム

  • 飛行物理 ＆ 電線への着地 ・羽ばたき時の浮遊感や慣性を物理シミュレート
    ・電柱や電線に着地して、スタミナを高速回復！
  • 楽しいフン爆撃ギミック ・歩行者や車にフンを命中させ、 楽しい悲鳴と高得点を獲得。
  • ハト・ショップでの育成 ・集めたパンでハトのスタミナやスピードを強化
    ・ゴールデンやサイバーなどのカスタムスキンを解放！

🛠️ 音声ファイル一切不要！

Web Audio API による 動的シンセサイザー

  • BGMや効果音（羽ばたき、ヒット音）は すべてコードで生成
  • 音声アセットの読み込みが無いため、 一瞬で超軽量に起動
  • HTML5 Canvas と JavaScript のみで高速・滑らかに動作

🤖 AIとの爆速共同開発

Google Antigravity との ペアプログラミング

  • 音響エンジニアリング Web Audio APIでの効果音・BGMのコード生成
  • ゲーム物理の調整 慣性のある飛行挙動や、電柱への当たり判定の最適化
  • デザイン最適化 ネオン調ダークテーマCSSとレスポンス画面調整
